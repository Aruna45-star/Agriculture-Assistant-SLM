# Agriculture Assistant SLM

An AI-powered agriculture chatbot built using FastAPI, React (Vite), Ollama, and the Qwen2.5:3B model.

## Features

- Agriculture-only chatbot
- English, Telugu, and Hindi support
- Voice input
- Dark mode
- React + Vite frontend
- FastAPI backend
- Ollama + Qwen2.5:3B local inference
